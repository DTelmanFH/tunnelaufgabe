Aufgabenstellung
  • Wenn Sie näher an einem Ausgang sind als an einem Telefon, wählen Sie das Telefon.
    -o-------X--- -> Mallorca

  • Befinden sich zwei Telefone in gleicher Entfernung, wählen Sie dasjenige, das näher an einem der Ausgänge liegt.
    -X-o-X-------- -> Barcelona

  • Wenn es wieder keinen Unterschied gibt, fahren Sie nach Mallorca.
    -X-o-X- -> Mallorca

  • Wenn Sie sich in einem Abschnitt mit einer Telefonnummer befinden, dann geben Sie ein Telefonsymbol aus
    ---(o/X)---x-- -> Telefon
